#  Projects 
## DorkMatrix
### Search Beyond Search Engines.

DorkMatrix is a fully open-source, self-hosted recon platform for bug hunters. Type a Google Dork query and instantly open it across Google, Bing, DuckDuckGo, Brave, and Yandex — each in its own browser tab, fully pre-filled.

## ReconVorteX
ReconVorteX streamlines reconnaissance by discovering, analyzing, and prioritizing attack surfaces through an automated multi-phase pipeline. It integrates industry-standard tools with intelligent Python-based analysis to produce comprehensive, high-quality recon results.
